drp-datepicker

A minimal demo of every feature in the package — open this file directly, no build step, no server.

1. Nepali-first picker type="bs" (default)

English date shown small, bottom-right of each cell. Holidays and a min/max range are set below.

<drp-datepicker placeholder="Select a date" min="2083-01-01" max="2083-02-30"></drp-datepicker>
(no selection yet)

2. English-first picker type="ad"

Same component, flipped: English calendar drives the grid, Nepali date shown small, bottom-right.

<drp-datepicker type="ad" placeholder="Select a date"></drp-datepicker>
(no selection yet)

3. Inline calendar + Devanagari digits

inline renders the calendar open in the page flow. digits="ne" starts it in Devanagari numerals (toggleable from the calendar footer either way).

<drp-datepicker inline digits="ne"></drp-datepicker>

4. First day of week first-day-of-week

Control which day starts the week. Default is Sunday (0); set first-day-of-week="1" for Monday, "6" for Saturday, etc.

first-day-of-week="0"
(Sunday — default)
first-day-of-week="1"
(Monday)
first-day-of-week="6"
(Saturday)

5. Display format format

Customize how the selected date appears in the input. Tokens: YYYY, YY, MMMM, MM, M, DD, D.

format="YYYY-MM-DD"
format="DD Month YYYY"

6. Custom disabled dates disabledDates

Block specific dates via the .disabledDates property. Below, Saturdays and a few holidays are disabled.

picker.disabledDates = [{date: '…'}, {date: '…'}]

Accessibility & positioning

      

        Keyboard: Arrow keys navigate the day grid, Enter/Space select,
        Page Up/Down switch months, Escape closes.

        Focus: Focus returns to the trigger on close. Tab cycles through
        panel controls.

        ARIA: role="dialog", aria-modal,
        aria-expanded, aria-selected, live region for screen readers.

        Auto-flip: The panel flips above the field when there isn't
        enough room below — try it by scrolling this card near the bottom of the viewport.
