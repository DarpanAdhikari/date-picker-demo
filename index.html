drp-datepicker

A minimal demo of every feature in the package — open this file directly, no build step, no server.

1. Nepali-first picker type="bs" (default)

English date shown small, bottom-right of each cell. Holidays and a min/max range are set below.

2. English-first picker type="ad"

3. Inline calendar + Devanagari digits

inline renders the calendar open in the page flow. digits="ne" starts it in Devanagari numerals (toggleable from the calendar footer either way).

4. Real <form> integration

name + required behave like a native input — picked up by FormData, blocks submit until filled.
